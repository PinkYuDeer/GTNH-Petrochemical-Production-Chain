# GTNH-Petrochemical-Production-Chain
一个用于计算GTNH中油的各种线路的工具。你只需要核对蒸馏数据.xlsx然后运行python脚本就可以生成原始数据。

也可以直接下载版本对应表格进行筛选、排序。表格已经进行整理，绿色为产量健康，红色为产量低于中位线，黄色为平均水平。
